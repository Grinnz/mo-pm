=encoding utf8

=head1 Name

Mo::builder - adds the features builder and default to Mo's has

=head1 Synopsis

    use Mo qw'has';
    has name1 => ( default => sub { 'Joe' } );
    has name2 => ( builder => 'name_lookup' );

=head1 Description

Loads the features L<Mo::builder> and L<Mo:default>.

=cut
